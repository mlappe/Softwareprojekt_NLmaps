this is the nmt part of our project

to run it use 
	sh runnmt.sh 
in this folder
this will train a new model and evaluate it on the test set

the output can be found in 
	./nmt/nmtout.mrl
		contains just the system mrls
	./nmt/out.txt
		contains the decoding step, linearized mrl and mrl
